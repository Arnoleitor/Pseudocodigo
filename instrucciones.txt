comandos terminal
-dir : te dira las carpetas que tienes en el directorio en el que esta la carpeta
-cd nombre carpeta : navega en dicha carpeta
-cd .. : vuelva atras

----proceso de subida de datos----
0-git status
1-git add "nombre archivo"(poner verde)
2-git commit -m +nombre "comilla"para reflejar cambio (mensaje)
3-git push para terminar de subir
-git pull para traer cambios

---cambio de ramas---
-git branch : saber las ramas que hay verde en la que estas
-git checkout +nombre rama para cambiar.
-git checkout -b+nombre crea rama y desplaza directamente
-para crear rama : git branch +nombre
-git branch -d +nombre rama para eliminar
-clonar :git clone +enlace

--Mezclar Ramas--
git merge +nombre rama que quieres traer
